# Zaawansowane Bazy Danych 2020

Wybrane zadania wykonane w ramach przedmiotu Zaawansowane Bazy Danych.

Dla każdego zadania plik `readme.md` zawiera opis zadania, pliki `report.tex` i `report.pdf` przygotowany przeze mnie raport.
Pozostałe pliki zawierają kod źródłowy, dane i wyniki.

Technologie:
* PostgreSQL 12 (+ cstore_fdw)
* Redis 5
* Ruby 2.7
